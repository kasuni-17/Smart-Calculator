# 🧮 Smart Calculator

A simple and modern calculator web application built using **HTML**, **CSS**, and **JavaScript**.
It supports basic arithmetic operations with a clean UI, keyboard input, and responsive design.

---

## ✨ Features

- ➕ Addition, Subtraction, Multiplication, Division
- 🧼 Clear (C) and Delete (⌫) functions
- ⌨️ Full keyboard support
- 📱 iPhone-style round buttons
- 🎨 Modern light UI design
- 🖥️ Real-time input display
- 🌄 Background image with soft overlay

## ⚙️ How to Run the Project

- Download or clone the project

```
git clone https: https://github.com/kasuni-17/Smart-Calculator.git
```

- Open the folder

```
cd Calculator
```

- Run `index.html` in any web browser

**👉 No installation required**

## ⌨️ Keyboard Controls

| Key       | Action            |
| --------- | ----------------- |
| 0-9       | Input numbers     |
| + - \* /  | Operators         |
| Enter     | Calculate result  |
| Backspace | Delete last digit |
| Escape    | Clear display     |
